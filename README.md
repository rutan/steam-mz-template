# RPG Maker MZ Template for Steam

- English: [README.en.md](./README.en.md)
- [更新履歴](./CHANGELOG.md)

## これはなに？

RPGツクールMZのSteam向けプロジェクトのテンプレートです。

RPGツクールMZで制作したゲームを Steam 向けにパッケージ化するためのアレコレが入っています。

## テンプレートのダウンロード
[リリースページ](https://github.com/rutan/steam-mz-template/releases) から最新版をダウンロードしてください。 
ファイル名が `steam-mz-template-○.○.○.zip` のようになっているものをダウンロードしてください。

詳しい使い方は同梱の説明書（`readme.html`） 、または [README.md](./template/README.md) をご覧ください。
