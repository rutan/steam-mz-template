# RPG Maker MZ Template for Steam

- Japanese: [README.md](./README.md)
- Changelog(Japanese): [CHANGELOG.md](./CHANGELOG.md)

## What is this?

This is a beginner-friendly template for packaging RPG Maker MZ projects for Steam using Electron.
The primary audience is Japanese users, but this English guide covers the minimum steps for global users.

## Downloading the Template

Please download the latest version from the [Releases page](https://github.com/rutan/steam-mz-template/releases).
Look for files named like `steam-mz-template-○.○.○.zip`.

For detailed instructions, refer to the included manual (`readme.en.html`) or the [README.en.md](./template/README.en.md) in the template folder.
