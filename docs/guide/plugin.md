# プラグインの導入

テンプレートに付属する `SteamPlugin.js` は、Steamテンプレートを使用するために必要なRPGツクールMZ用プラグインです。 **テンプレートを利用する場合は、このプラグインを必ず導入してください** 。

<div class="tip custom-block">

このプラグインは基本的にSteam以外の場所では何も起きません。そのため、同じゲームをSteam以外で公開する場合でも、プラグインを有効のままにしておいて問題ありません。

</div>

## 導入方法

ダウンロードしたテンプレートのフォルダ内に `plugins` という名前のフォルダがあります。この中に `SteamPlugin.js` が含まれています。

このプラグインを、あなたのRPGツクールMZプロジェクトの `js/plugins` フォルダにコピーしてください。

その後、『プラグイン管理』を開き、`SteamPlugin.js` を追加してください。

## プラグインの設定

プラグイン設定では `Steam App ID` の設定が必須です。

Steamworksの管理画面で、あなたのアプリケーションを作成した際に発行された `App ID` を入力してください。

それ以外の設定項目は必要に応じて変更してください。詳細は [プラグインのリファレンスページ](../reference/steam-plugin-about.md) を参照してください。
