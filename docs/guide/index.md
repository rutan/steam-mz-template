# はじめに

『RPG Maker MZ Template for Steam』は、RPGツクールMZで作成したゲームをSteamで公開するためのテンプレートプロジェクトです。

本テンプレートを利用することで、比較的容易にRPGツクールMZで作成したゲームをSteam向けにパッケージ化できます。

## 機能

- Electronを利用したパッケージ化
  - RPGツクールMZで作成したゲームを [Electron](https://www.electronjs.org/) を用いてパッケージ化します。
- Steamworks SDKの統合
  - [steamworks.js](https://github.com/ceifa/steamworks.js/) を利用して、Steamworks SDKを統合しています。これにより、Steamの機能（実績、クラウドセーブなど）をゲームに組み込むことが可能です。
- batファイルによるお手軽ビルド
  - 用意されたバッチファイルを実行するだけで、複雑なコマンド入力をすることなく、簡単にビルドが行えます。
- Steam連携用プラグインの提供
  - RPGツクールMZ用のSteam連携プラグインを提供します。ゲーム制作者はプラグインを設定するだけで、Steamとの連携が可能です。
