# ゲームのパッケージ化

実際にテンプレートを利用してゲームをパッケージ化する手順を説明します。

## (1) 作成したゲームをデプロイメントする

RPGツクールMZのメニューから `ファイル` → `デプロイメント` を選択します。

`プラットフォーム` で `ウェブブラウザ` を選択してください。

未使用ファイルの除外などは必要に応じて設定してください。

最後に `OK` をクリックして、デプロイメントを実行してください。

## (2) 出力されたゲームデータをテンプレートにコピーする

![データをコピーするイメージ図](/img/guide/package/copy.png)

出力されたゲームデータのフォルダ内にあるすべてのファイル・フォルダを、テンプレートの `app` フォルダ内にコピーしてください。

## (3) テンプレートの `command.bat` を実行する

テンプレート内にある `command.bat` をダブルクリックして実行してください。
すると、以下のようなメッセージが表示されます。

```
----------------------------------------
RPGツクールMZ Steam 向けテンプレート
----------------------------------------
1. ツールをセットアップ（ npm install ）
2. ゲームをデバッグ起動（ npm run dev ）
3. ゲームをパッケージ化（ npm run package ）
0. 終了

選択肢を入力してください:
```

まずは `1`を入力してEnterキーを押し、ツールのセットアップを行ってください。
一度セットアップが完了すれば、次回以降はこの手順をスキップして構いません。

完了したら、次は `2` を入力してEnterキーを押し、ゲームのデバッグ起動を行ってください。
Steamクライアントが起動し、ゲームが正しく動作するか確認できます。
ここでゲームが正しく動作しない場合は、ゲームデータのコピーやプラグインの導入が正しく行われているか確認してください。

動作に問題がなければ、最後に `3` を入力してEnterキーを押し、ゲームのパッケージ化を行ってください。この処理には数分かかる場合があります。

処理が完了すると、`out` フォルダ内にパッケージ化されたゲームが出力されます。
このフォルダ内のファイルをSteamworksにアップロードすることで、Steam上でゲームを公開できます。

Steamworksへのアップロード方法については、 [Steamworks のドキュメント](https://partner.steamgames.com/doc/sdk/uploading) を参照してください。
