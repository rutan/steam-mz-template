# 更新履歴

アップデート後はセットアップ（ npm install ）を再度実行してください。

# 2026/02/07

- テンプレートのライセンスを MIT No Attribution License (MIT-0) に変更しました
  - これにより利用時にテンプレート自体の著作権表示を行う必要がなくなりました
  - ただし、テンプレートに含まれるサードパーティ製ライブラリ等のライセンスには従う必要があります
- ビルド時に、サードパーティ製ライブラリ等のライセンス情報をまとめたファイルをゲームに含めるようにしました
  - ゲームのインストールフォルダ内に「THIRD_PARTY_LICENSES.txt」という名前で出力されます

## 2026/01/21

- Electron v.40.0.0 に更新しました
  - Node.js のバージョンは v.22 以上が推奨になります
- プラグインコマンドに「外部URLを開くコマンド」を追加しました
- ゲーム内で window.open を使用した際に、外部ブラウザで開くようにしました
- SceneManager.exit を使用した際に、ゲームを終了するようにしました
- 実験的機能として、プラグインヘッダーの出力機能を追加しました
  - 使用プラグインのヘッダーコメントを抽出して、まとめて出力します
  - 使用する場合はプラグイン設定で有効にしてください（デフォルトは無効です）
    - プラグインによっては正しく動作しない場合があります

## 2025/02/18

- RPGツクールMZ v.1.9.0 に対応しました
  - エディター設定の「JSON整形レベル」を変更した場合の動作に対応しました

## 2025/01/30

- SteamPlugin.js を更新しました
  - 以下の設定項目を追加しました
    - バージョン情報
    - 著作情報
- パッケージ化する際に exe ファイルにバージョン情報と著作情報を埋め込むようにしました
- パッケージ化する際に、ゲームに不要なテンプレート用のファイルが含まれないようにしました
